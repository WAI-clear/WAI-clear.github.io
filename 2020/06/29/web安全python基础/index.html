http协议介绍
  • 协议分类
    • HTTP1.0 80端口
    • HTTP1.1 80端口
  • 协议弊端和修正
    • 弊端： 明文传输不安全
    • 修在：使用HTTPS协议，加密传输数据
  • 请求方法：
    • HTTP1.0
      • GET
      • POST
      • HEAD
    • HTTP1.1&HTTPS:
      • PUT
      • DELETE
      • TRACE
      • CONNECT
      • PATCH
  • HTTP请求头
    • User-Agent: 主要是浏览器信息
    • Acccept-encoding: 浏览器接受的编码
    • Refer: 当前网页跳转来源
    • Location: 跳转目的地
    • WWW-Authenticate: 用于身份验证HTTP Basic等。
  • HTTP响应状态码：
    • 1XX:信息提示
    • 2XX: 请求成功
    • 3XX：重定向（请求的页面可能已经迁移，服务端跳转到新的页面地址）
    • 4XX: 客户端错误（发送了不存在的请求
    • 5XX: 服务端错误
